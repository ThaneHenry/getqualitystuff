# Resources

Useful external sources and services used by the project.

## Design

- [Google Fonts Icons](https://fonts.google.com/icons) - Icon library for interface design.
- [Google Fonts](https://fonts.google.com/) - Provides the Unbounded display typeface used by the site.
- [Flag Icons](https://flagicons.lipis.dev/) - Country flag icons used on listings.
- [Unsplash](https://unsplash.com/) - Source of example imagery used in the sample import data.

## Infrastructure

- [jsDelivr](https://www.jsdelivr.com/) - Delivers the Flag Icons stylesheet.
- [DreamHost](https://www.dreamhost.com/) - Production hosting and deployment target.
